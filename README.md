

**Nome do Programa:** Cadastro de Informações Pessoais

**Descrição:**
Este é um programa em C# que coleta e exibe informações pessoais do usuário. Ele solicita os seguintes dados:

1. Nome completo
2. Quantidade de quartos na residência
3. Preço de um produto
4. Último nome, idade e altura (na mesma linha)

O programa processa os dados inseridos, convertendo-os para os tipos apropriados (string, int, double) e formata a saída para números reais com duas casas decimais. Utiliza métodos de parsing com `CultureInfo.InvariantCulture` para garantir compatibilidade com diferentes formatos numéricos internacionais.

**Funcionalidades:**
- Entrada de dados via console
- Processamento de múltiplos tipos de dados
- Formatação de saída para valores monetários e decimais
- Separação de dados em vetor usando o método `Split()`

**Saída:**
O programa exibe todas as informações fornecidas pelo usuário formatadas de maneira organizada, seguindo o padrão especificado para números reais.
